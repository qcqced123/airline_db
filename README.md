# airline_db
This is a simple web application that allows users to book flights or etc.

## Project Structure
**MVT(Model/View/Teamplate)** Pattern is used in this project. The project structure is as follows:
```
├── app/
│      ├─ __init__.py  # default run file
│      ├─ models.py
│      ├─ forms.py
│      ├─ views/
│      │   └─ main_views.py
│      ├─ static/
│      │   └─ style.css
│      └─ templates/
│            └─ index.html
└── config.py
